# PatchUp Web Archive

Archived Next.js frontend for PatchUp. The active client is now the Expo mobile app in `apps/mobile`.

## Run locally

This folder is kept for reference and migration history. It is no longer part of the active workspace scripts.

## Highlights

- App Router
- Type-safe forms with React Hook Form + Zod
- Zustand auth store
- Role-based dashboards
- Reusable Tailwind UI primitives
